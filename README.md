# Sentiment-Analysis
The idea: Website Sentiment Analysis Tool - A multilingual text sentiment analysis tool that allows users to input any text with status updates, accurately interpreting the content in the original language, displaying results using doughnut graphs and progress bars (% for each sentiment), and saving analysis history for later review.
Frontend: HTML / JS
Backend:Node.js + Express
NLP:Amazon Comprehend
Step 1: Download the project source code to your computer or clone the repository from GitHub.

Step 2: Open the terminal in the project directory.

Step 3: Install the necessary packages using the command
npm install

Step 4: Start the server using the command
node app.js

Step 5: Open your browser and access the address
http://localhost:3000

Step 6: Enter the text into the input field and click the parse button to send a request to AWS Comprehension and receive the results.
