# Sentiment-Analysis
Ý tưởng: Website Sentiment Analysis Tool - Công cụ phân tích cảm xúc văn bản đa ngôn ngữ cho phép người dùng nhập bất kỳ đoạn văn bản nào với các dòng trạng thái có thể đọc nội dung đúng ngôn ngữ gốc, hiển thị kết quả bằng biểu đồ doughnut + progress bars (% cho từng cảm xúc) và có thể lưu lịch sử phân tích để xem lại.
